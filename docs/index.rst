SeisGo Documentation
====================

**SeisGo** is an open-source Python package for seismic data analysis, ambient noise seismology,
seismic monitoring, and related workflows. It provides a unified, high-level interface covering
everything from raw data download through cross-correlation, stacking, dispersion analysis,
anisotropy measurement, and velocity model clustering.

- **GitHub**: https://github.com/xtyangpsp/SeisGo
- **License**: MIT
